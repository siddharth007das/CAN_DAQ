# CAN_DAQ
The CAN-DAQ (Controller Area Network Data Acquisition) system is an open-source, cost-effective platform for automotive data acquisition and analysis. The system supports standard CAN baud rates of 125 kbps, 250 kbps, 500 kbps, and 1 Mbps, and is compatible with any CAN Database Container (DBC) file for automatic signal decoding and interpretation.
